最关注的事


1. OpenAI 发布 GPT-5.5 并将模型与 Codex 托管 Agent 引入 AWS


OpenAI 同时发布了 GPT-5.5 和 GPT-5.5 Instant 两款模型，并宣布 OpenAI 模型、Codex 以及 Managed Agents 正式登陆 AWS 平台。这意味着企业开发者可以在 AWS 基础设施上直接调用 OpenAI 的编码 Agent 能力，大幅降低了将 agentic coding 集成到现有云工作流的门槛。对 Agentic SE 领域而言，这是模型能力与云平台深度绑定的标志性事件。


来源：OpenAI models, Codex, and Managed Agents come to AWS


2. Cloudflare 允许 Agent 自主创建账户、购买域名并部署服务


Cloudflare 宣布 AI Agent 现在可以通过集成 Stripe 支付，自主完成从创建 Cloudflare 账户、购买域名到部署项目的完整流程。这是 Agent 从"写代码"走向"端到端交付"的重要一步——Agent 不再只是辅助开发，而是能独立完成从基础设施采购到上线的全链路操作。这对 agentic software engineering 的自主性边界是一次实质性扩展。


来源：Agents can now create Cloudflare accounts, buy domains, and deploy


3. GitHub Copilot 转向按用量计费


GitHub 宣布 Copilot 将从固定订阅制转向基于使用量的计费模式，同时调整了个人版计划。这一商业模式变化反映了 AI 编码工具正在从"功能附加"演变为按消耗计量的基础设施服务，也暗示着 Agent 模式下 token 消耗量的不可预测性正在倒逼定价模型变革。


来源：GitHub Copilot is moving to usage-based billing
