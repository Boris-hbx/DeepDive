### 1. Supermaven 加入 Cursor：AI 代码补全工具深度整合
Cursor 正式整合以极速补全著称的 Supermaven，标志 AI 编程工具的竞争进入“编辑器+补全引擎”一体化时代。此举将大幅提升代码生成的响应速度与上下文理解能力，对开发者日常工作流及 AI coding 工具格局产生直接影响。 [原文](https://cursor.com/blog/supermaven)

### 2. Symphony：OpenAI 开源编排规范，将 Issue Tracker 变为永不休息的 Agent
该规范通过 Codex 把项目管理工具转化为可长期运行的 Agent 系统，自动推进任务、减少上下文切换，直接体现 SPEC 驱动开发与 SDD 理念。它为“代码仓库即 Agent 中枢”提供了标准化路径，有望重塑 AI 驱动的软件交付流程。 [原文](https://openai.com/index/open-source-codex-orchestration-symphony)

### 3. Copilot 应用科学团队的 Agent 驱动开发一手经验
GitHub 工程师公开了使用 coding agents 自动化自身工作的完整旅程，涵盖提示设计、人机协同边界和实际收益。这份实战报告为团队落地 AI 驱动开发（SDD）提供了可复制的模式，验证了 agents 能真正接管非平凡工程任务。 [原文](https://github.blog/ai-and-ml/github-copilot/agent-driven-development-in-copilot-applied-science/)
