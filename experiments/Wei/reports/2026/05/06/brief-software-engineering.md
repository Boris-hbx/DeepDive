### 1. OpenAI 开源 Symphony：用 Issue Tracker 驱动 Agent 编排的 SPEC
OpenAI 发布开源规范 Symphony，可将 GitHub Issues 等任务跟踪器直接转化为常驻 AI Agent 系统，自动将需求拆解为可执行步骤。这标志着 SDD（规范驱动开发）从概念走向落地——工程师只需写好 Issue 规格，Agent 即可自主完成开发与验证，大幅降低上下文切换成本。[原文](https://openai.com/index/open-source-codex-orchestration-symphony)

### 2. Supermaven 加入 Cursor，AI 编码技能栈再升级
以极速代码补全闻名的 Supermaven 宣布加入 Cursor 团队。两者整合后，Cursor 有望在推理速度与上下文感知能力上实现突破，直接增强 AI 代码补全、审查与重构的一体化体验，让“AI 原生编程环境”的边界进一步扩展。[原文](https://cursor.com/blog/supermaven)
