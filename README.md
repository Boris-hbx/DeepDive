# DeepDive

洞察探索（Insight Exploration）应用。前期 Web，后续可能扩展到 Android / HarmonyOS / iOS。

本项目同时是团队探索 Claude Code 协作模式的载体——code agent、spec、skill、人机协作、多 agent 协作等。

## 目录

- [`CLAUDE.md`](./CLAUDE.md) — 团队与 Claude Code 协作的规则（**必读**）
- [`spec/`](./spec) — 产品与功能规格
  - [`product.md`](./spec/product.md) — 一页纸产品定义
  - [`_template.md`](./spec/_template.md) — 功能 spec 模板
- [`docs/decisions.md`](./docs/decisions.md) — 关键决策记录

## 上手

1. 读 `CLAUDE.md`
2. 读 `spec/product.md`
3. 新功能从复制 `spec/_template.md` 开始

## 当前阶段

起步期。以下尚未确定，遇到再讨论：技术栈、部署方式、分工、是否引入多 agent / skill 等高级模式。
